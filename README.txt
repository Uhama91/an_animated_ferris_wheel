### README in English:

#### Ferris Wheel Animation (freeCodeCamp Guided Project)

A simple yet captivating animation of a Ferris Wheel built using HTML and CSS. This project is based on a guided project from freeCodeCamp and demonstrates the power and capabilities of CSS animations and transforms.

**Features:**
- Animated rotation of the wheel.
- Smoothly animated cabins, with changing colors.
- CSS `@keyframes` to control animation sequences.
- Use of `transform-origin` to set the axis of rotation.

**How to Run:**
1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.

**Technologies Used:**
- HTML5
- CSS3

### README en Français:

#### Animation de la Grande Roue (Projet Guidé de freeCodeCamp)

Une animation simple mais captivante d'une grande roue réalisée en HTML et CSS. Ce travail est basé sur un projet guidé de freeCodeCamp et démontre la puissance et les capacités des animations et des transformations CSS.

**Fonctionnalités:**
- Rotation animée de la roue.
- Cabines animées en douceur avec des changements de couleurs.
- Utilisation de `@keyframes` pour contrôler les séquences d'animation.
- Utilisation de `transform-origin` pour définir l'axe de rotation.

**Comment exécuter:**
1. Clonez ou téléchargez le dépôt.
2. Ouvrez `index.html` dans votre navigateur web préféré.

**Technologies utilisées:**
- HTML5
- CSS3
